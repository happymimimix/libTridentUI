## libTridentUI
**An extremely lightweight Chrome Embedded Framework alternative for legacy operating systems.**

**Works perfectly on Microsoft Windows XP SP3!**

Want to develop a browser-based desktop application for legacy operating systems? Try libTridentUI!

Built on Internet Explorer’s Trident engine, offering absolute top-tier compatibility!!!

You can learn libTridentUI in only a few minutes because it provides you with the most modern and easy to use API set you'll likely ever encounter on a Trident-based library.
